一、数据库

1.1、database（DB）：数据库：按照数据结构来组织，存储和管理数据的仓库。

Database Management System （DBMS）：对数据进行管理的软件系统

1.2、近年数据库使用情况：DB-ENGINES

1.3、MySql是一个开源的关系型数据库管理系统。

1.4、安装：MySQL Community Downloads

配置密码等

添加环境变量：安装目录\MySQL\bin

1.5、主要目录结构：

  • bin：可执行文件，如mysql.exe
  • include：存储头文件，如mysql.h
  • lib：存储库文件
  • share：存储错误信息、字符集文件
  • data：用于放置日志文件和数据库
  • upload：存放文件的导入导出
  • my.ini：配置文件
  • 其中最后三个目录在C:\ProgramData\MySQL\MySQL Server 8.0路径下
